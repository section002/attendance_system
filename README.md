# Attendance

## 環境構築

### 1.リポジトリのクローン

```sh
git clone https://github.com/section002/attendance_system.git
```

### 2.ビルド・依存関係のインストール

```sh
./mvnw clean install
```

### 3.開発サーバーの起動

```sh
./mvnw spring-boot:run
```

ブラウザで http://localhost:8080 へアクセス
